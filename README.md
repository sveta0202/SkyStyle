# SkyStyle
This project was created as part of a InnoHackathon in 2026.

<p><strong>Твой личный стилист, который знает, какая сегодня погода</strong></p>
 
## О проекте

SkyStyle — удобный веб-сервис, который помогает подбирать стильные образы из вещей, которые уже есть в вашем гардеробе, с учётом реальной погоды за окном.

Однажды добавив свой гардероб, вы каждый день получаете продуманные рекомендации, что надеть сегодня — практично, красиво и без стресса.

## Функционал 

- Подключение к API погоды (Openweather), на странице /home можно задать город и в блоке выдаст соответвующую погоду в реальном времени. (по умолчанию стоит Москва)
- Генерация образов (работает и без задания ваших вещей в профиле), достаточно выставить желаемые параметры и ИИ сгеренирует описание образа и добавить в ответ краткую сводку по погоде в городе, который вы указывали и учтенный гардероб.
- После генерации образ сохраняется в отдельную вкладку по пути /outfits-page с датой генерации и описанием образа.
- Реализована вся логика регистации и входа на базе PostgreSQL.

## Запуск проекта

Проект можно запустить на localhost на версии коммита 2779ec8dc5a874d9bcad2ca228a8440ccbba2a07 (debug) с помощью команды **docker compose up --build** из корневой директории.

Последняя версия 78ba59bc6fac6ab52f34a1bc903f158e9969ea63 (update Dockerfile) настроена под продакшн: https://calm-flexibility-production-2cf5.up.railway.app (ссылка актуальна до 24.08.2026). 

*Переключение вкладок работает только на ПК-версии. На мобильной для переключения страниц меняйте путь в url сайта:

/ - лендинг\
/reg - регистрация\
/auth - вход\
/home - главная страница\
/outfits-page - страница с созданными образами\
/generate - генерация образа\
/profile - профиль пользователя
